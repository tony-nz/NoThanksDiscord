NoThanksDiscord

A BetterDiscord plugin that hides the Nitro, Shop, and Quests tabs — plus other upsell clutter — from your sidebar. No upsells, no clutter, no seasonal sounds. Just the stuff you actually use.

⬇️ Download plugin ★ View on GitHub

# What it hides

Nitro tab On

Removes the Nitro tab from the home sidebar.

Shop tab On

Removes the Shop tab from the home sidebar.

Quests tab On

Removes the Quests tab, including its various element variants.

Gift button On

Removes the gift-a-Nitro icon next to the message box.

Boost upsells On

Removes “Boost this server” banners and prompts.

Nitro upsells On

Removes “Get/Try Nitro” prompts in the emoji picker, profiles, and more.

App launcher Opt-in

Removes the activities / app-launcher button by the message box.

Server discovery Opt-in

Removes the “Discover” compass at the bottom of the guild list.

Active Now Opt-in

Removes the friend-activity panel on the Home page.

Seasonal sounds On

Stops Halloween / Winter sound packs from overriding your soundpack.

# Install

  1. Install BetterDiscord if you haven't already.
  2. Download NoThanksDiscord.plugin.js.
  3. Drop it into your BetterDiscord plugins folder (paths below), or use Settings → Plugins → Open Plugins Folder.
  4. In Discord, open Settings → Plugins and enable NoThanksDiscord.
OSPlugins folder
macOS~/Library/Application Support/BetterDiscord/plugins
Windows%AppData%\BetterDiscord\plugins
Linux~/.config/BetterDiscord/plugins

# How it works

UI hiding injects CSS targeting stable attributes (href, aria-label) or stable class-name prefixes ([class*="questsButton_"]) — never full class names. Discord rotates the hash on its class names every build, but the prefix and these attributes stay put, so the selectors survive where class-based ones break.

Seasonal sounds are handled by patching Discord's SoundpackStore: when getSoundpack() reports a seasonal pack (halloween, winter_holiday), the plugin returns classic instead — so seasonal audio stops overriding your choice, while any non-seasonal pack you picked yourself is left alone.

Tip: every item is an independent toggle in Settings → Plugins → NoThanksDiscord → ⚙️. Hide everything, or just the bits that bug you.
